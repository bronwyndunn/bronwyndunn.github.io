My Skills

Curate utilizes a Ruby on Rails backend, a PostgreSQL database, and ReactJS with a Redux architectural framework on the frontend. I created custom SQL queries on ActiveRecord for search functionality via query strings. I utilized Cloudinary's API to allow users to upload photos and store images in the cloud to reduce server load.

Flexbox Flamingos is a frontend application built with React and Javascript. I created a pseudo-console where users directly manipulate the DOM by typing in CSS commands, and implemented click handlers that check for correct commands and allow users to move to new stages.

I teamed up with three other engineers to build Inheritage, a full-stack web application that utilizes a Python and Django backend with a ReactJS and Redux frontend. We developed a RESTful API for creating and displaying user profiles. We also optimized database queries using Django's ORM, and integrated the Recharts library to display dynamic graphs. We also incorporated Google Maps API to display user ethnic locations.
